## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   (This step must be run once before starting the app)
   python rag.py

3. Start backend:
   uvicorn main:app --reload

4. Open:
   http://127.0.0.1:8000

## Features
- Symptom-based case retrieval
- FHIR bundle export
- CSV data viewer
- Frontend UI
